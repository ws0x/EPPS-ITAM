# Legacy Data Migration Directory

Place your Snipe-IT backup files here:
1. **Snipe-IT SQL Dump**: E.g., `backup.sql` or `mysqldump.sql`.
2. **CSV Files**: Exported tables if SQL dump is not preferred.

Once the files are placed here, the migration script will be configured to read, parse, and ingest them directly into the new Supabase Postgres database.
